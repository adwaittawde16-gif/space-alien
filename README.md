# 👾 Space Alien

A browser-based space shooter game built with vanilla HTML, CSS, and JavaScript — no libraries or frameworks required.

---

## 🎮 Demo

Open `space.html` in any modern browser to start playing instantly.

---

## 🚀 Features

- Space-themed shooter gameplay
- Pure vanilla JavaScript — no dependencies
- Runs entirely in the browser (no server needed)
- Lightweight and fast

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Structure | HTML5 |
| Styling | CSS3 |
| Game Logic | Vanilla JavaScript |

---

## 📁 Project Structure

```
space-alien/
├── space.html   # Game entry point
├── space.css    # Styles and visual design
├── space.js     # Game logic and mechanics
└── README.md
```

---

## ▶️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/adwaittawde16-gif/space-alien.git
   ```

2. **Open the game**
   ```bash
   cd space-alien
   open space.html
   ```
   Or simply double-click `space.html` to open it in your default browser.

No build step, no install — just open and play!

---

## 🕹️ How to Play

- Use your keyboard or mouse to control your spaceship
- Shoot down incoming aliens before they reach you
- Survive as long as possible and rack up your score
